# 随机选择人员系统

从 Excel 人员名单中随机抽取人员，支持临时抽选与拖地排班两种模式。

## 快速开始

### 环境要求

- Python 3.10+
- 依赖库：`flet`、`pandas`、`openpyxl`

```bash
pip install flet pandas openpyxl
```

### 启动

```bash
python random_selector_ui.py
```

### 数据文件

程序默认读取 `config/data/PersonnelList.xlsx`。Excel 文件应包含以下列：

| 序号 | 姓名 | 学号 | 班级 | 性别 |
|------|------|------|------|------|

首次使用或更换数据文件时，通过菜单 **文件 → 打开数据文件** 选择你的 Excel 文件。

## 功能说明

### 两种抽选模式

| 模式 | 人数 | 保存状态 | 适用场景 |
|------|------|----------|----------|
| 临时模式 | 可自定义 | 不保存 | 临时点名、课堂提问 |
| 拖地模式 | 固定 3 人 | 保存到文件 | 排班轮值、值日安排 |

### 抽选操作

1. 在窗口中选择 **临时模式** 或 **拖地模式**
2. 临时模式需要输入抽选人数，拖地模式自动选择 3 人
3. 点击 **开始选择** → 逐行动画显示结果
4. 抽选结果自动出现在下方，人数较多时分两栏显示

### 回溯替换

如果选中人员因请假等情况无法参与：

1. 在回溯面板的输入框中输入该人员的 **学号**
2. 点击 **回溯替换**
3. 系统自动从剩余人员中随机选一人替换

### 查看人员

- **查看所有人员**：展示全量名单及选中状态（绿色=已选，红色=未选）
- **查看未选择人员**：仅展示尚未被选中的名单

## 配置与数据存储

所有配置和数据均存放在 `config/` 目录：

```
config/
├── ABOUT.md          # 关于信息
├── settings.json      # 用户配置（数据文件路径、种子）
├── icon.ico           # 程序图标
└── data/
    ├── PersonnelList.xlsx   # 默认数据文件
    └── exports/             # 导出结果目录
```

设置文件 (`settings.json`) 会在更改数据文件或修改选项时自动更新。

---

## Seed 种子功能

在 **工具 → 选项** 中启用固定种子后，使用相同种子值、相同数据文件进行抽选，结果完全一致。适用于：

- 测试验证
- 需要复现某次抽选结果的场景
