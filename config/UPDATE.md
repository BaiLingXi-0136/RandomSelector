# 版本更新记录

## v4.4.2（2026-07-23）

- 新增程序自动/手动检测更新功能，从 GitHub 拉取最新版本号比对
- 选项菜单增加"启动时自动检查更新"开关，可持久化设置
- 窗口标题追加版本号（如 `随机点名系统 v4.4.2`），标题栏右侧实时显示更新状态
- 手动检查更新时弹出加载等待窗口，后台线程执行不阻塞 UI
- 支持 hosts 文件 DNS 劫持检测，自动回退 GitHub CDN 真实 IP 直连
- 新增 `update_check.py` 模块（零额外依赖，纯标准库）
- 帮助界面新增"用户反馈"按钮，点击跳转至 GitHub Issues 页面
- 新增 `config/update.md`，用于记录版本更新内容

## v4.4.1（2026-07-12）

- 修复刷新功能，使其可视化
- 实时显示区突出展示当前选中人员姓名
- 修复回溯替换时可能重复选中同一人的问题
- 导出抽选结果时包含随机种子信息
- 新增 Traceback 反馈窗口与日志持久化

## v4.3.0

- 项目更名并增强多项功能
- 完成多文件架构重构与常量全面提取

## v4.2.0 及更早

- 项目初始开发阶段
- 将单一主文件拆分为多模块架构，新增 PyInstaller 打包支持
- 添加默认人员名单 Excel 模板文件
