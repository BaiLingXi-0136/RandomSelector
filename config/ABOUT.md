# 随机点名系统

**版本 4.3**

---

本程序用于从 Excel 人员名单中随机抽取人员，支持临时抽选与拖地排班两种模式。

### 主要功能

- **临时模式**：自定义 1~20 人，不保存状态，适合临时随机点名
- **拖地模式**：每日一次，选择 3 人并保存状态，适合排班轮值
- **回溯替换**：支持对已抽选人员实时替换（如请假等情况）
- **文件占用监测**：自动检测 Excel 文件是否被占用，及时提醒并保护数据
- **Seed 种子**：固定随机种子使结果可复现
- **实时校验**：人数输入、学号回溯均有即时反馈
- **单实例保护**：防止重复启动
